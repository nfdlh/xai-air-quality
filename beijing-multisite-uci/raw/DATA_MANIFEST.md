# Beijing Multi-Site Air Quality Dataset

## Source
- UCI ML Repository
- URL: https://archive.ics.uci.edu/dataset/501/beijing+multi+site+air+quality+data
- Time: March 2013 - February 2017
- Rows: 420,768 hourly observations
- Stations: 12 sites

## Columns
| Column | Description |
|--------|-------------|
| PM2.5 | Target variable (µg/m³) |
| PM10 | PM10 (µg/m³) |
| SO2 | Sulfur dioxide |
| NO2 | Nitrogen dioxide |
| CO | Carbon monoxide |
| O3 | Ozone |
| TEMP | Temperature (°C) |
| PRES | Pressure (hPa) |
| DEWP | Dew point (°C) |
| RAIN | Precipitation (mm) |
| wd | Wind direction |
| WSPM | Wind speed (m/s) |
| station | Station name |
